# SMG Motors - Next-Gen Electric Mobility

![SMG Motors Banner](/public/assets/hero-scooter.png)

## üöÄ Overview

**SMG Motors** is a cutting-edge frontend application designed for the modern electric vehicle (EV) industry. This project showcases a premium, high-performance landing page and customer portal for SMG Electric Scooters. It features a cinematic user experience, interactive 3D-like elements, and a seamless booking flow.

The application is built with **Next.js 14**, **TypeScript**, and **Tailwind CSS**, ensuring top-tier performance, SEO optimization, and a responsive design that looks stunning on all devices.

## ‚ú® Key Features

-   **Cinematic Hero Section**: Immersive full-screen video backgrounds and high-fidelity product imagery.
-   **Interactive Model Showcase**: Dynamic 360-degree style views and detailed spec breakdowns for different scooter models (Sprint, Blaze, Volt Pro).
-   **Smart Accordion UI**: "Why Choose Electric" section with smooth animations and detailed educational content.
-   **Service Subpages**: Dedicated pages for Roadside Assistance, Insurance, and Extended Warranty with custom branding.
-   **Dealer & Order Tracking**: Integrated flows for finding dealers and tracking vehicle delivery status.
-   **Responsive Design**: Fully optimized for mobile, tablet, and desktop experiences.

## üì¶ Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/abhishekk-y/SMG-Frontend.git
    cd SMG-Frontend
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  **Open the app:**
    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
